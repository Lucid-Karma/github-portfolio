How To Play

Use left click to rotate roads. Connect all of them and revive city. You can use right mouse button down to see different angles.

If there are others on the network, you can also play against them using the Challenge button.

If you have enough point (50pts), you can buy solve rights and use them whenever you want. This is the only way to finish a level if you struggle with it, but if you want to cheat, each time you restart the game, there is going to be a different puzzle for the current level.
